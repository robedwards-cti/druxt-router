# DruxtRouterMixin

The DruxtRouterMixin Vue.js mixin provides the props required for a Router Druxt wrapper component.

Kind: global mixin
Example

<template>
  <div>
    <slot />
  </div>
</template>

<script>
import { DruxtRouterMixin } from 'druxt-router'

export default {
  mixins: [DruxtRouterMixin]
}
</script>
